Descripción del proyecto


Consiste en una base de datos iterativa para manejar los proyectos que se realizan en el Decanato de Investigación de UNIBE. (publicación, proyectos, congresos), la cual implementa un Web service que se encarga gestionar las diferentes entidades ya mencionadas.


Equipo de Desarrollo
